# 123TV

- Proxify content from 123tv
## Installation

Install with git

```bash
  git clone https://github.com/ParrotDevelopers/123TV.git
  cd 123TV
  npm install
```
    
## Usage

### Running
```bash
npm run start
```
Then open ```http://ip:port/playlist.m3u```  

### M3U
Create user in Admin CP
```
http://ip:port/username/password/playlist.m3u
```
